再看STL标准模板库

根据 Acwing 算法基础课整理。

再看STL标准模板库

参考：https://cplusplus.com/reference

pre：pair

定义：
pair<int,string> p;

初始化：
第一种：p=make_pair(10,"abc");
第二种：p={10,"abc"};

重要字段：
第一个元素：p.first
第二个元素：p.second

其他特性：
支持比较运算（以 first 为第一关键字，second 为第二关键字，按字典序）
拓展到三个值 pair<int,pair<int,int>>

string 字符串

常用函数：
size() length() 返回字符个数
empty() 返回是否为空
clear() 清空
substr() 返回字串（第一个参数：index；第二个参数：length）
c_str() 返回字符串对应的字符数组头指针（转换为字符数组）

其他特性：
字符串拼接 +=赋值操作

vector 变长数组

倍增的思想：
系统为 vector 分配空间所需时间，与空间大小无关，与申请次数有关
vector 变长后复制
故向 vector 插入数据时，申请空间时间复杂度总共 O(logn)，复制时间复杂度平均 O(1)

定义：
vector<int> a 变长数组 a
vector<int> a(10) 长度为10的变长数组 a
vector<int> a(10,3) 长度为10的变长数组 a，并且所有元素都为 3

常用函数：
size() 返回元素个数
empty() 返回是否为空
clear() 清空
begin() 头迭代器
end() 尾迭代器
front() 返回第一个数
back() 返回最后一个数
push_back() 向尾插入
pop_back() 从尾弹出

其他特性：
支持 [] 随机寻址
支持比较运算（按字典序）

遍历方法：
vector<int> a;
第一种：for (int i=0;i<a.size();i++) cout << a[i] << ' ';
第二种：for (vector<int>::iterator i=a.begin();i!=a.end();i++) cout << *i << ' ';
for (auto i=a.begin();i!=a.end();i++) cout << *i << ' ';
第三种：for (auto x : a) cout << x << ' ';

stack 栈

定义：
stack<类型> 变量名;

常用函数：
size() 返回元素个数
empty() 返回是否为空
无 clear
top() 返回栈顶元素
push() 向栈顶插入
pop() 从栈顶弹出

queue 队列

定义：
queue<类型> 变量名;

常用函数：
size() 返回元素个数
empty() 返回是否为空
无 clear 替代方案：变量名 = queue<int>();
front() 返回队头元素
back() 返回队尾元素
push() 向队尾插入
pop() 从队头弹出

遍历方法：
queue<int> myqueue;

priority_queue 优先队列（堆）

定义：
默认是大根堆
大根堆：priority_queue<类型> 变量名;
小根堆：priority_queue<类型,vector<类型>,greater<类型>> 变量名

常用函数：
size() 返回元素个数
empty() 返回是否为空
无 clear
top() 返回堆顶元素
push() 向堆里插入
pop() 从堆顶弹出

deque 双端队列（升级版的 vector）

缺点：较慢

定义：
deque<类型> 变量名;

常用函数：
size() 返回元素个数
empty() 返回是否为空
clear() 清空
begin() 头迭代器
end() 尾迭代器
front() 返回第一个数
back() 返回最后一个数
push_back() 向尾插入
pop_back() 从尾弹出
-
push_front() 向头插入
pop_front() 从头弹出

其他特性：
支持 [] 随机寻址

set, multiset, map, multimap 基于平衡二叉树（红黑树）

功能：动态维护有序序列

定义：
set, map 无重复元素，multiset, multimap 可以有重复元素
set<类型> 变量名;
multiset<类型> 变量名;
map<类型,类型> 变量名;
multimap<类型,类型> 变量名;

常用函数：
size() 返回元素个数
empty() 返回是否为空
clear() 清空
begin() 头迭代器
end() 尾迭代器

-set, multiset-
insert() 插入一个元素
count() 返回某个元素出现的次数
find() 查找一个元素（返回迭代器）
lower_bound(x) 返回大于等于 x 的最小的数的迭代器
upper_bound(x) 返回大于 x 的最小的数的迭代器
erase() (1) 输入是一个数 x，删除所有 x；(2) 输入是迭代器，删除这个迭代器对应的值

-map, multimap-
insert() 插入一个元素 - pair
count() 返回某个元素出现的次数 - key
find() 查找一个元素 - key（返回迭代器）
lower_bound(x) 返回大于等于 x 的最小的数的迭代器 - key
upper_bound(x) 返回大于 x 的最小的数的迭代器 - key
erase() (1) 输入是 pair，删除所有此键值对；(2) 输入是迭代器，删除这个迭代器对应的值

其他特性：
支持迭代器的 ++ 和 –，返回前驱和后继
对于 map 和 multimap，有map[键]=值，时间复杂度O(logn)

unordered_set, unordered_multiset, unordered_map, unordered_multimap 基于哈希表

与set, multiset, map, multimap类似，但增删改查的时间复杂度为O(1)
因为无序，不支持lower_bound(x) upper_bound(x)和迭代器的 ++ 和 –

bitset 压位

bitset 是位操作的对象，值只有 0 或 1

定义：
bitset<个数> 变量名

支持：
~ 取反，& 逻辑与，| 逻辑或，^ 逻辑异或
<< 左移，>> 右移
==，!=
[]

常用函数：
count() 返回某一个数的个数
any() 判断是否至少有一个 1
none() 判断是否全为 0
set() 把所有位赋为 1
reset() 把所有位赋为 0
set(k,v) 把第 k 位赋为 v
flip() 把所有位取反
flip(k) 把第 k 位取反
